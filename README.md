# PCC Sensors

A collection of sensors based on ESP8266/ESP32 built using [ESPHome](https://esphome.io/).
Below is a list of existing of sensors already developed:

- [Environment Sensor](env/README.md)

## Installation

1. Clone this repo.
2. Create an virtualenv with `python3 -m virtualenv .venv`.
3. Install requirements with `python3 -m pip install -r requirements.txt`.
4. Now follow the README for each sensor.
